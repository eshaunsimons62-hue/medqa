# 🏥 MedQA — Software Testing & Validation Suite

> A standalone web app that demonstrates professional QA practices — automated test execution, defect tracking, test coverage metrics, and downloadable FDA-style validation PDF reports.

---

## 📋 Overview

MedQA simulates the rigor of a regulated medical software environment. It provides a structured pipeline for running automated tests against defined software requirements, capturing defects, and generating audit-ready PDF reports aligned to **IEC 62304** medical device software standards.

Built as a portfolio project to demonstrate QA engineering and software validation skills.

---

## ✨ Features

- ✅ **Automated Test Execution** — Runs test cases against defined software requirements with pass/fail tracking
- 🐛 **Defect Tracking** — Log and manage defects with severity levels and status lifecycle
- 📄 **FDA-Style PDF Report** — Generate and download a structured validation report referenced against IEC 62304
- 📊 **Dashboard** — Visual overview of test coverage and pass/fail metrics

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed
- npm

### Installation

```bash
git clone https://github.com/eshaunsimons62-hue/MedQA.git
cd MedQA
npm install
npm start
```

## 📄 IEC 62304 Alignment

The generated validation report references the following IEC 62304 clauses:

| Clause | Description |
|---|---|
| 5.1 | Software development planning |
| 5.5 | Software unit implementation and verification |
| 5.7 | Software release |
| 8.2 | Problem and modification analysis |
| 9.8 | Software problem resolution records |
